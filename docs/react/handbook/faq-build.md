## Как в JSX написать комментарий? {#how-can-i-write-comments-in-jsx}

```js
<div>
  {/* Это комментарий */}
  Привет, {name}!
</div>
```

```js
<div>
  {/* А это
  многострочный комментарий. */}
  Привет, {name}!
</div>
```
